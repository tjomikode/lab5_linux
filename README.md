# lab5_linux

##Arguments that my programm supports:

1.NO ARGUMENT -> (-h or --help) 
2.OPTIONAL ARGUMENT -> (-s or --size)
	*size without value -> (-s 100 or --size 100)
	*size with value -> (-s100 or -size100 or --size=100)
3.REQUIRED ARGUMENT -> (-f or --file)
	*file = example:
		>-f example
		>-fexample
		>--file example
		>--file=example
